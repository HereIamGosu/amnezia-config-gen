Генератор конфигураций AmneziaWG

Конфигурацию можно сгенерировать для любых устройств. Она содержит заранее внесённые маршруты для Discord.

Файл планировщика доступен только для Windows.

Дополнительная информация

Инструкция по установке и настройке

  1. Скачивание и установка AmneziaWG:
    Перейдите по ссылке для скачивания и загрузите установочный файл для Windows. Установите по стандартному пути "C:\Program Files\...".
  2. Генерация конфигурационного файла:
    После установки, нажмите кнопку «Сгенерировать» слева для создания конфигурационного файла. Скачивайте файл в папку «Загрузки», куда будет смотреть AmneziaWG.
  3. Загрузка и запуск файла планировщика:
    Скачайте .bat-файл для планировщика и запустите его от имени администратора.

Теперь AmneziaWG будет автоматически подключаться к нужному конфигу при входе в систему 🥸

Сгенерированная конфигурация настроена так, что она перенаправляет только трафик, связанный с Discord. Это достигается за счёт указания диапазонов IP-адресов Discord в переменной AllowedIPs в файле конфигурации.

Если вы хотите, чтобы весь ваш интернет-трафик проходил через туннель AmneziaWG, вы можете удалить строку с AllowedIPs из конфигурационного файла.

Также вы можете заменить диапазоны IP-адресов на свои. Например, вы можете указать диапазоны для других сервисов (YouTube, Spotify, Twitch или любых других). Список IP-адресов для различных сервисов можно найти на ресурсе rockblack.pro.
