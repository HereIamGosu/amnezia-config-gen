Генератор конфигураций AmneziaWG

Конфигурацию можно сгенерировать для любых устройств. По умолчанию весь трафик идёт через туннель. Пресеты маршрутов и DNS можно задать в модальном окне по кнопке ⚙ рядом с кнопками генерации.

Файл планировщика доступен только для Windows.

Инструкция по установке и настройке

  1. Скачивание и установка AmneziaWG:
    Перейдите по ссылке для скачивания и загрузите установочный файл для Windows. Установите по стандартному пути "C:\Program Files\...".
  2. Генерация конфигурационного файла:
    После установки выберите слева «Сгенерировать (Legacy)» или «Сгенерировать AmneziaWG 2.0» в зависимости от версии клиента. Сохраните файл в папку «Загрузки», куда будет смотреть AmneziaWG.
  3. Загрузка и запуск файла планировщика:
    Скачайте .bat-файл для планировщика и запустите его от имени администратора.

Теперь AmneziaWG будет автоматически подключаться к нужному конфигу при входе в систему 🥸

Дополнительная информация

По умолчанию весь трафик идёт через туннель (AllowedIPs = 0.0.0.0/0, ::/0). В модальном окне настроек (кнопка ⚙) можно выбрать DNS-сервер и отметить пресеты — тогда в AllowedIPs попадут только подсети выбранных пресетов.

Также вы можете вручную заменить диапазоны IP-адресов на свои. Список IP-адресов для различных сервисов можно найти на ресурсе rockblack.pro.

Настройки конфигурации

DNS-сервер
Пресеты маршрутов (AllowedIPs)

Отметьте пресеты — при генерации их подсети (IPv4 и IPv6) будут запрошены и объединены в AllowedIPs. Если ничего не выбрано, туннель охватывает весь трафик (0.0.0.0/0, ::/0).
